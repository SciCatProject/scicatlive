# [Openapigenerator](https://github.com/OpenAPITools/openapi-generator)

OpenAPIGenerator is a tool that generates API client libraries, server stubs, and documentation
from OpenAPI specifications.
In SciCat, it is used to generate the SDKs from the running backend in DEV mode.

It is accessible at `http://openapigenerator.localhost`, when the [proxy service](../../../../../proxy) is enabled.

## Configuration options

No custom configuration options are currently set up for this service.

## Default configuration

Same as upstream default configs.

## Enable additional features

No additional features.
